# gitmoji

提交时使用表情是个好主意。访问 gitmoji.dev在新窗口打开 来查看更多内容。

git commit -m ":bug: Fix ..."

这样我们可以看到 🐛 Fix ... 这样的信息。各种不同表情的含义可以直接在上面的网站上看到。

# 扩展功能

可以安装 gitmoji-cli在新窗口打开 来尝试更花哨的功能：

npm i -g gitmoji-cli

如果需要引用该项目的所有表情，可以：

npm i gitmojis

这样可以得到包含所有表情信息的数组：

import { gitmojis } from 'gitmojis'

console.log(gitmojis)

[
  {
    emoji: '🎨',
    entity: '&#x1f3a8;',
    code: ':art:',
    description: 'Improve structure / format of the code.',
    name: 'art',
    semver: null
  },
  // ...
]
